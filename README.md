# Econverse Front-End Challenge

Repositorio da solucao do teste para vaga Front-End da Econverse.

## Stack

- React 18
- TypeScript
- Sass
- Vite

## Como rodar localmente

```bash
git clone https://github.com/destruidorluck/teste-front-end.git
cd teste-front-end\site
npm install
npm run dev
```

App em desenvolvimento: `http://localhost:3000`

## Como testar e validar

```bash
npm run type-check
npm run lint
npm run test
npm run build
```

## Storybook

```bash
npm run storybook
npm run build-storybook
```

## Deploy (GitHub Pages)

```bash
npm run deploy
```

URL publica:

- https://destruidorluck.github.io/teste-front-end/

## Requisitos do desafio

- React + TypeScript: OK
- Consumo de JSON da API oficial: OK
- Modal ao clicar no produto: OK
- Sass como pre-processador: OK
- Sem bibliotecas UI (Bootstrap/Foundation): OK
- SEO basico + HTML semantico: OK
